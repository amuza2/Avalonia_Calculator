## Features

## Quick Start

```bash
git clone https://github.com/amuza2/Avalonia_Calculator.git
cd Avalonia_Calculator
dotnet run --project src/CalculatorAvaloniaUI
```

## Build & Test

```bash
dotnet build
dotnet test
```

## Publish

```bash
./publish.sh  # Select platform
```
